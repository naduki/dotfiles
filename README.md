# dotfiles -- My NixOS Settings

- Version : Unstable
- Desktop : Cinnamon
- Editor : VSCode
- Shell : Bash
- Terminal: Wezterm
- Browser : Brave

# Caution
Cinnamon Wayland だと Wezterm のメニューバーが表示されないので、ウィンドウを移動できない。またフルスクリーンにしたら戻せなくなる
代替は、Alacrittyだったけど、Rio Terminal も良さそう。
